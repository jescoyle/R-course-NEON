Setting up your computer

R and RStudio

R and RStudio are separate downloads and installations. R is the underlying statistical computing environment, but using R alone is tricky for beginners. RStudio is a graphical integrated development environment (IDE) that makes using R much easier and more interactive. You need to install R before you install RStudio. In the sections below are the instructions for installing R and R Studio on your operating system.

If you don’t have R installed

If you have never used R or RStudio on your computer, then you probably don’t have R installed.

  1. Visit the CRAN website
  2. In the first box labeled “Download and Install R”, click on the link for the operating system on your computer (e.g. Windows, Mac or Linux)

R for Windows

  1. Click “base”. Then on the next page, click “Download R.x.x.x for Windows” (x.x.x are the version number)
  2. Save the .exe file to a location on your computer (e.g. the Downloads folder).
  3. Open and run the .exe file you just downloaded. This will install and set up R on your computer.

R for Mac

  1. Click on the .pkg file for the latest R version that will work on your computer.
  2. Save this file somewhere on your computer (e.g. Downloads)
  3. Open the downloaded .pkg file to install R
  4. It is also a good idea to install XQuartz. Follow the installation instructions on that website. R will work without XQuartz, but you may find that you need it later if certain packages do not install properly.

R for Linux

  1. Choose your distribution of Linux.
  2. Follow the instructions on the website.

RStudio for all operating systems

  1. Go to the RStudio download page
  2. The website should automatically detect your operating system and provide the correct installer download as a large button. If you need a different version, scroll down the page and select the installer download that matches your operating system.
  3. Save the file somewhere on your computer (e.g. Downloads).
  4. Open the downloaded file to launch the RStudio installer and follow the instructions in the prompts.
  5. Once it is installed, open RStudio to make sure it works and you don’t get any error messages.

If you do have R and RStudio installed

  • Open RStudio, and click on “Help” > “Check for updates”. If a new version is available, quit RStudio, and download the latest version for RStudio.
  • To check which version of R you are using, start RStudio and the first thing that appears in the console indicates the version of R you are running. Alternatively, you can type sessionInfo(), which will also display which version of R you are running. Go on the CRAN website and check whether a more recent version is available. If so, please download and install it. You can check here for more information on how to remove old versions from your system if you wish to do so.
