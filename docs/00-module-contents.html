Lesson Contents

This page will give an overview of episodes within this tutorial and learning objectives accomplished in each. It will correspond to the computing skills checklist used in the course.

Core Chapters

Chapter 1. Navigating your computer

Learning Objectives

In this lesson learners will:

  • describe the structure of the filesystem on a computer
  • use a command line interface to navigate the file system
  • describe the relationship between R and RStudio (an IDE)
  • set up an RStudio project

Computational Skills

After completing this lesson, learners will be able to use a Unix terminal to:

  1. navigate between directories
  2. view files in a directory
  3. make, delete, rename and move files

Chapter 2. Intro to R and RStudio

Learning Objectives

In this lesson learners will:

  • discover why so many scientists use R
  • describe the contents of the windows in an RStudio session
  • install and load packages in R
  • write and save a code script in a project folder
  • read and view tabular data from a file
  • use the console for mathematical calculations
  • state the rules for assigning variables
  • work with vectors
  • learn how to get help and troubleshoot error messages

Computational Skills

After completing this lesson, learners will be able to use R and RStudio to:

  1. execute simple mathematical calculations
  2. assign and manipulate variables
  3. assign, subset and manipulate data in a vector
  4. execute vector algebra
  5. write and save a code script
  6. write comments in a code script
  7. read a data table from a file
  8. access and understand a function’s documentation page

Chapter 3. Data organization in spreadsheets

Learning Objectives

In this lesson learners will:

  • describe best practices in working with data tables
  • understand the structure of a text-based data table file
  • use metadata to understand the contents of a data table
  • utilize quality control features of spreadsheet software to generate clean data

Computational Skills

After completing this lesson, learners will be able to
  1. follow best practices in formatting data tables
  2. document steps of a data analysis using metadata and a README file
  3. import and export text-based data tables using spreadsheet software

Chapter 4. Data structures and operations

Learning Objectives

In this lesson learners will:

  • describe the main data structures and types in R
  • manipulate and extract values from data frames using base R

Computational Skills

After completing this lesson, learners will be able to
  1. differentiate between vectors, arrays, lists and data frames in R
  2. differentiate between numeric, character, factor and logical data types in R
  3. view the structure and class of an object in R
  4. subset data frames and vectors in R
  5. write logical test expressions using ==,!=, >, <, and %in%

Chapter 5. Vectorization and functions

Learning Objectives

In this lesson learners will:

  • describe what vectorized functions are and how to use them in R
  • explain how functions work and define and our own functions in R
  • explain why we should divide programs into small single-purpose functions

Computational Skills

After completing this lesson, learners will be able to
  1. define and use a function with multiple arguments in R
  2. define default values for function arguments in R
  3. write checks on argument conditions within a function in R
  4. differentiate between missing values (NA) and empty values (NULL) in R
  5. use R to source code from a script

Chapter 6. Simple graphs with ggplot2

Learning Objectives

In this lesson learners will:

  • use ggplot2 to generate single-panel graphics with one data layer and one statistic layer
  • manipulate the aesthetics of a ggplot using different colors, shapes, and lines
  • use different geometries to change the way data is displayed in a ggplot
  • export graphs in a specific format and size

Computational Skills

After completing this lesson, learners will be able to use the ggplot2 package in R to:

  1. create a scatter plot with a linear regression line
  2. create a line graph
  3. create a bar chart with error bars
  4. create a stacked bar chart
  5. map aesthetics to data values
  6. write axis labels with mathematical notation
  7. export graphs to a vector-format image with specific dimensions

Chapter 7. Controlling program flow

Learning Objectives

In this lesson learners will:

  • use conditional statements to make data-dependent decisions in R
  • use loops to repeat things in R
  • write expressions containing multiple logical statements in R

Computational Skills

After completing this lesson, learners will be able to use R to:

  1. execute if-else statements
  2. repeat steps using for and while loops
  3. evaluate expressions using & (‘and’) and | (‘or’)

Chapter 8. Complex graphs with ggplot2

Learning Objectives

In this lesson learners will:

  • create multi-panel graphics with multiple data layers using ggplot2
  • describe how position scales control the way axes are displayed in ggplot2
  • describe how themes control the way plots are displayed in ggplot2
  • select inclusive color palettes that are visible to people with impaired color vision
  • change the default labels in a legend in ggplot2
  • add reference lines to a plot in ggplot2

Computational Skills

After completing this lesson, learners will be able to use the ggplot2 package in R to:

  1. display data in separate panels of a graph using facets
  2. add multiple data layers to a plot and change the order in which they are layered
  3. change the scale, limits and breaks on an axis
  4. change the default fonts, colors, and sizes using a theme
  5. change a discrete and continuous color scale
  6. change the text labels in a legend
  7. add horizontal and vertical reference lines to a plot

Chapter 9. Data manipulation in the tidyverse

Learning Objectives

In this lesson learners will:

  • differentiate between longer and wider data formats and explore when to use each
  • use functions in the dplyr and tidyr R packages to combine, subset, summarize and change the layout of data frames
  • write multi-step data manipulations by composing functions together with pipes

Computational Skills

After completing this lesson, learners will be able to
  1. use R to join two tables using a key
  2. use R to select columns from a table using matching criteria
  3. use R to filter rows from a table
  4. use R to create new columns with mutate
  5. use R to calculate summary statistics across factors with group_by and summarize
  6. use R to change the format of a table from long to wide or wide to long with pivot functions from the tidyr package
  7. compose functions with pipes (%>%)
